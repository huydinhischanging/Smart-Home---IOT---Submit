sequenceDiagram
    autonumber
    actor User as 👤 User (Web/Mobile)
    participant Nginx as 🔀 Nginx
    participant Flask as 🐍 Flask API
    participant JWT as 🔑 JWT Module
(flask-jwt-extended)
    participant DB as 🗄️ PostgreSQL

    rect rgb(232, 245, 233)
        Note over User,DB: 📋 ĐĂNG KÝ (Register)
        User->>Nginx: POST /api/auth/register
{username, email, password}
        Nginx->>Flask: forward request
        Flask->>DB: check email đã tồn tại?
        DB-->>Flask: not found
        Flask->>Flask: bcrypt.hash(password, salt=12)
        Flask->>DB: INSERT user (hashed_password)
        DB-->>Flask: user created
        Flask-->>User: 201 {message: "registered"}
    end

    rect rgb(224, 242, 241)
        Note over User,DB: 🔐 ĐĂNG NHẬP (Login)
        User->>Nginx: POST /api/auth/login
{email, password}
        Nginx->>Flask: forward request
        Flask->>DB: SELECT user WHERE email=?
        DB-->>Flask: user row
        Flask->>JWT: bcrypt.check(password, hash)
        JWT-->>Flask: ✅ valid

        Flask->>JWT: create_access_token(user_id, expires=15min)
        Flask->>JWT: create_refresh_token(user_id, expires=30days)
        Flask->>DB: store refresh_token hash (revocation list)
        Flask-->>User: 200 {
  access_token: "eyJ...",
  refresh_token: "eyJ...",
  expires_in: 900
}
        User->>User: lưu tokens vào storage
    end

    rect rgb(243, 229, 245)
        Note over User,DB: 🛡️ GỌI API BẢO MẬT (Protected Request)
        User->>Nginx: GET /api/devices
Authorization: Bearer eyJ...
        Nginx->>Flask: forward + header
        Flask->>JWT: verify_jwt_in_request()
        JWT->>JWT: decode & check (exp, iss, alg=HS256)
        JWT-->>Flask: ✅ identity = user_id
        Flask->>DB: SELECT devices WHERE user_id=?
        DB-->>Flask: device list
        Flask-->>User: 200 [{device_code, status...}]
    end

    rect rgb(255, 243, 224)
        Note over User,DB: ♻️ LÀM MỚI TOKEN (Refresh)
        User->>Flask: POST /api/auth/refresh
Auth: Bearer refresh_token
        Flask->>JWT: verify refresh token
        JWT-->>Flask: ✅ valid, not revoked
        Flask->>DB: check revocation list
        DB-->>Flask: not revoked
        Flask->>JWT: create_access_token(user_id)
        Flask-->>User: 200 {access_token: "new_eyJ..."}
    end

    rect rgb(255, 235, 238)
        Note over User,DB: 🚪 ĐĂNG XUẤT (Logout)
        User->>Flask: POST /api/auth/logout
Auth: Bearer refresh_token
        Flask->>DB: INSERT revoked_tokens (jti, expires_at)
        DB-->>Flask: ok
        Flask-->>User: 200 {message: "logged out"}
        Note over User: xoá tokens khỏi storage
    end
